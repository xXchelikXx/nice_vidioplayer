# Видео плеер
Этот проект построен с помощью библиотеки [video-player-jslib](https://github.com/devmanorg/video-player-jslib).

## Как использовать
Тут всё работает как и на других сайтах: кнопка для паузы/проигрывания, включить/выключить звук и открыть на весь экран. 

Здесь можно установить ссылку на другое видео:




Так же можно сразу зайти на сайт по [ссылке](https://xxchelikxx.github.io/nice_vidioplayer/dist)
