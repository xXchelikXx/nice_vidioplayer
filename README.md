# Видео плеер
Этот проект построен с помощью библиотеки [video-player-jslib](https://github.com/devmanorg/video-player-jslib).

## Как использовать
Тут всё работает как и на других сайтах: кнопка для __паузы/проигрывания__, __включить/выключить__ звук и __открыть на весь экран__. 

Здесь, в `src`, можно установить ссылку на __другое__ видео:
